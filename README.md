> **IT Startup Tycoon** is a 2D management simulation application built in Java Swing where players run their own software company. Through a clean user interface, players manage a corporate budget, hire developers from a dynamic job market, and complete contracts, all while the underlying code actively demonstrates core Object-Oriented Programming principles.

---

## ✨ Key Features & Mechanics

| Feature | Description |
| :--- | :--- |
| ⏳ **Time & Resource Management** | Advance the days to progress projects, but watch your budget as daily expenses and employee salaries are deducted. |
| 👔 **Dynamic HR Market** | Open the HR department window to view a list of developers with different skill levels (Junior/Senior), working speeds, and salary expectations. |
| 📝 **Contract Fulfillment** | Accept software projects of varying difficulties and assign them to your team to earn money and keep the company afloat. |
| 📊 **Interactive Dashboard** | Monitor your company's live stats, active employees, and ongoing projects directly from the main office interface. |

---

## 🎮 Controls & Gameplay

The application is entirely controlled via the mouse through a standard, intuitive graphical user interface.

* **Title Screen:** Click **"Start Game"** to initialize the simulation.
* **Dashboard - Progression:** Click **"Next Day"** to advance time, trigger work progress, and process salary deductions.
* **Dashboard - Expansion:** Click **"Hire Employees"** to open the HR window.
* **HR Department:** Select an available developer from the list and click **"Hire"** to add them to your team (this automatically deducts the hiring fee from your budget).

## ⚙️ Installation & Setup

**Prerequisites:**
* Java Development Kit (**JDK**) 17 or higher.
* An IDE that supports Java (e.g., IntelliJ IDEA, Eclipse, or VS Code).

**Execution Steps:**

1. Clone this repository to your local machine: `git clone [https://github.com/v-humll/projekt_Huml_2026](https://github.com/v-humll/projekt_Huml_2026)`
2. Open the project in your preferred IDE.
3. Locate the `Main.java` file in your source folder.
4. Run the `main` method to launch the Title Screen.
